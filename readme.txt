== Description ==

OnDestek websitelerinize 2dk içerisinde kolayca entegre edilen bir canlı destek sistemidir.Sistemi kullanabilmek için öncelikle [Kayıt Ol](http://ondestek.com/kayit-ol?utm_campaign=wordpress&utm_medium=wp_market&utm_source=plugin.wordpress.org) sayfasından ücretsiz kaydınızı yapabilirsiniz.
Sonrasında sizin için oluşturulan kodu eklenti aktif hale geldiğinde açılan bölüme ekleyebilirsiniz.

== Installation ==

Kurulum oldukça basittir

1. Eklenti dosyalarını `/wp-content/plugins/ondestek` klasörüne kopyalayın ya da direkt olarak wordpress eklenti bölümünden kurulumunu yapın.
2. Wordpress yönetim paneli içerisinden Eklentiler bölümünden eklentiyi etkinleştirin.
3. Ayarlar > Ondestek menüsüne girerek, OnDestek kodunuzu ekleyerek, etkinleştir tuşuna basınız.
4. Eğer henüz kodunuz yok ise, http://ondestek.com/kayit-ol sayfasına giderek kaydınızı gerçekleştiriniz. Sistem sizin için bir kod üretecektir.

== Frequently Asked Questions ==

= 7 GÜNLÜK DENEME NE DEMEK? =
7 gün boyunca, Profesyonel Paketin bütün özelliklerini kullanabilirsiniz. Yapmanız gereken sadece üye olup, sizin için oluşturulan kodu sitenize eklemek. Kredi kartı bilgilerinize ihtiyacımız yok. Deneyin, kesinlikle memnun kalacaksınız.


= 7 GÜNLÜK DENEME PAKETİ BİTTİĞİNDE NE OLACAK? =
Deneme paketiniz bittiğinde, size e-posta ile bildireceğiz. Sistemimizi satın almayı düşünüyorsanız, ücretli bir paket seçip, ödemenizi yapabilirsiniz. 
Eğer herhangi bir paket satın almazsanız, üyeliğiniz otomatik olarak ücretsiz pakete düşürülecektir.


= FATURALANDIRMA DÖNGÜSÜ NASIL? = 
İlk ödemenizi nasıl yaparsanız, Faturalandırma Döngünüz o şekilde devam etmektedir.

Örneğin 1 yıllık paket satın alırsanız Fatura Döngünüz de 1 yıl olacaktır. İstemeniz halinde, bize ulaşırsanız, faturalandırma döngünüzü değiştirebiliriz.


= BİR HESAP İLE BİRDEN FAZLA SİTE YÖNETEBİLİR MİYİM? =
Evet. Sistemimiz ile bu mümkün, ancak kurumsal ya da profesyonel paket müşterilerimiz bu hizmetten yararlanabiliyorlar. Kurumsal müşterilerimiz 10 adet websitesine kadar, Profesyonel paketi kullanan müşterilerimiz ise, sınırsız site ekleme olanağına sahipler.


= MÜŞTERİ TEMSİLCİSİ NEDİR? =
Müşteri Temsilcisi, panele giriş yaparak ziyaretçileriniz ile sohbet ederek onlara destek olan kişidir.


= ÜCRETLENDİRME NASIL OLUYOR? =
Fiyatlarımız, Müşteri Temsilcisi x Paket Fiyatı şeklinde ücretlendirilmektedir. Örneğin 3 adet müşteri temsilcisi almak istediniz. Paket fiyatı aylık 11$ ise, 11x3=33$ olarak ücretlendirilir.


= İSTEDİĞİM ZAMAN PAKET DEĞİŞTİREBİLİR MİYİM? =
Evet istediğiniz zaman paketinizi değiştirebilirsiniz. İsterseniz üst pakete, isterseniz de alt pakete geçiş yapabilirsiniz.


= İSTEDİĞİM ZAMAN MÜŞTERİ TEMSİLCİSİ EKLEYEBİLİR MİYİM? =
Evet istediğiniz zaman müşteri temsilcisi/temsilcileri ekleyebilirsiniz. Eklediğiniz müşteri temsilcisi/temsilcileri için olan ekstra ücreti ödemeniz durumunda eklenebilmektedir.
